# Sistema de Hamburgueria 🍔

Este é um projeto de desenvolvimento de um sistema de hamburgueria em desenvolvimento, estou desenvolvendo para relembrar alguns conceitos de [Tailwindcss](https://tailwindcss.com/)

## Funcionalidades
- Adicionar Produtos no carrinho para compra.
- Realizar compras de produtos.

## 🛠️ Tecnologias Utilizadas (Até o momento)
- HTML
- tailwindcss
- OBS: Esse projeto está em desenvolvimento é a parte de interação que será utilizando o JavaScript ainda está em desenvolvimento.
